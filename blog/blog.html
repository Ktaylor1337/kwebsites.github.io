Hello, world!

May 9th 2017

I wrote my first line of code on Christmas day 2016. Since then I’ve made three websites, using HTML5, CSS3 and Javascript.

I think the best word to describe the experience so far is ‘empowering.’ Learning to code is empowering. You learn to code and you suddenly have this tool set which you can use to build things that you couldn’t before. I’ve never been particularly practical (I’m better at thinking than doing), but coding allows you to be a digital engineer and actually make things that function from almost nothing. That’s empowering.

Here’s the first website that I built, using Bootstrap, HTML5, CSS3, and some Javascript.

I was following a Udemy course and deviated quite quickly from their theme so that I could get creative and put my own personal stamp on things. I rushed into it head first without understanding how to properly lay out the page. This meant there was a lot of ad-hoc adjustments to margins and padding to try to get everything to align. The code itself is stuck together here and there without any proper organisation. It violates best practice. But it’s something, and I was and still am quite proud of my first ever website.

Next I thought I would have a go at designing an ‘actual’ website - so I took my local GP’s website which you can see here ( disclaimer: it’s a UX/UI nightmare), and had a go at re-presenting the information in a new design. This website is almost finished - I just need a techy friend to help me with the API key.

My most recent website is a revision/upgrade to my first website - the one you are on now, - this time built using best practice principles (and without a framework), and some other web design tips and tricks I have picked up over the last few months.
